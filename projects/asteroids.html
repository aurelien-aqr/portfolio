Asteroids WebGL (Projet de BUT3)

  • Compétences déployées : WebGL (GLSL, buffers, shaders), JavaScript (ECS), Physique 2D, Particules, HUD, Gestion des collisions.
  • Lien : Site web produit

Réalisation d’un clone du jeu Asteroids en WebGL, dans lequel on pilote un vaisseau pour détruire des astéroïdes.

Fonctionnalités principales :

  • Rendu via WebGL : interface HTML minimale, tout est dessiné dans un seul canvas.
  • Système de particules pour les explosions, les débris.
  • Contrôles clavier : rotation du vaisseau, accélération, tir, hyperspace aléatoire.
  • Gestion des collisions : astéroïdes divisés en plus petits morceaux, débris du vaisseau en cas de destruction.
  • HUD dessiné en WebGL : score, niveau, nombre de vies, message "GAME OVER".

Aspects techniques :

  • Architecture ECS : position, velocity, collision, rendering, lifetime, etc. permettant un code modulaire.
  • Shaders GLSL : un vertex shader pour la transformation 2D, un fragment shader pour la couleur (et alpha).
  • Physique simplifiée : wrap de l’écran (si on sort à droite, on réapparaît à gauche), inertie, etc.
  • Particules : random speed, rotation, fade-out, lifetime.

Ce projet a été réalisé en solo dans le cadre de mon BUT3, pour le cours de Sensibilisation à la programmation multimédia. Il m’a permis de découvrir le développement de jeux vidéo en 2D, et de mettre en pratique les concepts de WebGL, de shaders, de particules, de physique 2D, et de gestion des collisions.
