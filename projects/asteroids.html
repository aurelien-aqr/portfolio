Asteroids WebGL (Projet de BUT3)

  • Compétences déployées : WebGL (GLSL, buffers, shaders), JavaScript (ECS), Explosions 2D, Particules, Rendu sans HTML/CSS, Physique simplifiée (collisions).

Réalisation d’un clone du jeu Asteroids entièrement géré en WebGL, dans lequel on pilote un vaisseau pour détruire des astéroïdes, accumuler des points et gérer un système de vies/respawn.

Fonctionnalités principales :

  • Rendu intégral via WebGL : aucune interface HTML, tout est dessiné dans un seul canvas.
  • Système de particules avancé pour les explosions, les débris (taille, alpha, interpolation de couleurs, rotation).
  • Contrôles clavier : rotation du vaisseau, accélération (affichage flamme), tir, hyperspace aléatoire.
  • Gestion des collisions : astéroïdes divisés en plus petits morceaux, débris du vaisseau en cas de destruction.
  • HUD dessiné en WebGL : score, niveau, nombre de vies, message "GAME OVER".

Aspects techniques :

  • Architecture ECS : position, velocity, collision, rendering, lifetime, etc. permettant un code modulaire.
  • Shaders GLSL : un vertex shader pour la transformation 2D, un fragment shader pour la couleur (et alpha).
  • Interpolation de couleurs et alpha pour les explosions : débris passant du jaune vif au gris fumée.
  • Physique simplifiée : wrap de l’écran (si on sort à droite, on réapparaît à gauche), inertie, etc.
  • Particules : random speed, rotation, fade-out, lifetime, explosion plus réaliste.

Ce projet a été réalisé dans un contexte de BUT3, pour illustrer une approche "sans HTML/CSS" en WebGL pur. Il m'a permis de comprendre les bases du shading, de manipuler la boucle de rendu et de mettre en œuvre un système de particules personnalisable pour les explosions d’astéroïdes.
