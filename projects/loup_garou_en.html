Online Werewolf Game - Play Anywhere, No Cards Needed (Personal Project)

  • Skills used: Web development, HTML, CSS, JavaScript, PHP, Databases, SQL.
  • Link: Live website

Development of a web application that allows users to play Werewolf online without physical cards, with an interface optimized for mobile devices. The website supports multiplayer games by letting one player create a session and invite others to join. The game master can manage participants and assign roles. The idea for this project came from an evening with friends when we wanted to play but did not have the cards with us. I decided to build a website that would make the game playable anywhere, without cards, on both phones and computers.

This was my first dynamic web project involving a database and JavaScript. I completed it independently before formally studying these topics in class. To overcome technical challenges, I relied on online tutorials and AI-assisted problem solving.

Key Features:

Join a game: Players can join a session by entering their name. They can then access their assigned role, hide it from view if needed, and read a description of that role.

Game management: The game master can manage players by assigning roles, removing participants, or randomly distributing selected roles.

Browse role details: Users can also view descriptions of all Werewolf roles without having to join a game.
