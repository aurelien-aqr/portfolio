Nuit de l'Info 2022

  • Compétences déployées : Travail d'équipe, Création d'un site web (HTML, CSS, Javascript), Création d'un jeu vidéo (Python, RenPy)
  • Liens : Site web produit | Jeu MSTCity

La Nuit de l’Info est une compétition nationale qui réunit étudiants, enseignants et entreprises pour travailler ensemble sur le développement d’une application web. Cet événement implique : Choix des défis à réaliser, Direction de la création du jeu vidéo à produire, Prises d'initiatives au cours de la nuit.

Le thème portait sur la prévention sexuelle. Il nous était demandé de programmer une application web ludique pour informer les jeunes (et les adultes) sur la santé sexuelle. Nous avons donc créé un site regroupant plusieurs jeux sur les IST et leurs risques. Nous avons notamment créé un jeu vidéo sur RenPy, de type visual novel, qui est un jeu de "drague" dans lequel le joueur doit faire les bons choix pour ne pas attraper d'IST.

Nous avons obtenu la 3e place sur 518 participants au classement national.

Au sein d'un groupe de 7, j'ai réalisé la majeure partie du jeu MSTCity.
