Asteroids WebGL (Bachelor Year 3 project)

  • Key skills used: WebGL (GLSL, buffers, shaders), JavaScript (ECS), 2D physics, Particles, HUD, Collision management.
  • Link : Live website

Development of a clone of Asteroids in WebGL, where you control a spaceship to destroy asteroids.

Main features:

  • Rendering with WebGL: minimal HTML interface, everything is drawn in a single canvas.
  • Particle system for explosions and debris.
  • Keyboard controls: ship rotation, acceleration, shooting, random hyperspace.
  • Collision system: asteroids split into smaller pieces, and ship debris on destruction.
  • HUD drawn in WebGL: score, level, lives count, and "GAME OVER" message.

Technical aspects:

  • Architecture ECS : position, velocity, collision, rendering, lifetime, etc. allowing modular code.
  • Shaders GLSL : a vertex shader for 2D transformations, un fragment shader for color (and alpha).
  • Simplified physics : screen wrapping (if you leave the right side, you reappear on the left), inertia, etc.
  • Particles : random speed, rotation, fade-out, lifetime.

This project was completed solo during my third year, for the Multimedia Programming course. It helped me discover 2D game development and apply WebGL, shaders, particles, 2D physics, and collision management concepts.
